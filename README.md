# Trabalho de desenvolvimento de software para web

Nome do Software Web: GreedStore

## Requisitos Funcionais
- Cadastrar Usuário - Qualquer usuário
- Manter decks - Admins → Criar deck, Excluir deck, Alterar deck, Consultar deck, baixar lista de cards do deck(em pdf), compartilhar deck.
- Mantes Usuários do site - Admins → Pode Adicionar, excluir, alterar e consultar usuários.
- Manter cards dos Usuários - Admins → Manter decks, manter favoritos, manter interesses de compra
- Manter Favoritos - Usuário → Criar favoritos, Excluir favoritos, Alterar Favoritos, consultar favoritos
- Listar todas as cartas - Usuário
- Pesquisar por nome - Usuário
- Filtrar por tipo - Usuário
- Filtrar por preço - Usuário
- Visualizar detalhes de cada carta - Usuário
- Visualizar média de preço dos cards - Usuário
- Criar Deck - Usuário
- Criar deck personalizado por IA - IA própria criadora de decks(Premium)
- Buscar lojas vendedoras - Usuário
- Fazer análise de mercado com IA - Usuário (Premium)
- Definir alarme de preço para carta específica - Usuário
- Esquecer Senha - Usuário
